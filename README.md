## рҹҺ® How to Play

### Solo
1. Open the app (web or APK)
2. Enter your **Groq API key** in the sidebar вҖ” it will be saved automatically
3. Choose a **topic** (or type a custom one)
4. Set **difficulty** and **word count**
5. Hit **Generate** вҖ” copy any word and use it as a charade

### Team
1. Go to the **TEAM** tab
2. Add all **player names**
3. Choose a **topic**, **difficulty**, and **timer length**
4. Hit **Start Game** вҖ” players take turns acting out words with the countdown running
5. Results and rankings appear after all rounds are complete
